# 상표 출원 가능성 예측 서비스 
<br>

**1️⃣  프로젝트 소개**

**유사군코드**와 **상표명**을 **입력**했을 때, 같은 상품의 상표명 데이터 내에서 **유사도가 높게 나타나는 상표명들**과 **가장 높은 유사도 확률**을 확인해 원하는 상표명이 출원 가능한지 판단할 수 있는 웹 서비스를 구축한다. 이후 추가적인 검색 옵션을 제공할 예정이다.
